----------------------------------------
부경대학교 공식 홈페이지 데이터를 크롤링하여 사용자가 원하는 교수님 정보(전공, 연구실명, 전화번호, 이메일)에 대해 응답. 또 사용자가 직접 입력한 강의평가를 감성평가모델을 이용해 긍정/부정을 판단한 후 응답데이터에 추가하여 사용자에게 강의평 정보를 제공

1. 사용자에게 정보를 원하는 교수님의 이름을 입력받으면 전역변수에 해당 교수명을 저장 후 해당 교수님의 어떤 정보를 원하는지 질문을 반환.
    
     1-1. 사용자가 입력한 값이 데이터베이스에 존재하지 않을 경우  “교수님 정보가 없습니다. 다시 입력해주세요” 응답 반환
    
2. 사용자가 ‘전공'/'연구실'/'이메일'/'전화번호'/'강의평' 중 원하는 정보를 입력하면 전역변수로 지정된 교수님의 데이터 중 사용자가 입력한 값에 해당하는 정보를 응답으로 반환
    
     2-1. 만약 사용자가 입력한 값이 위의 항목에 없으면
    
     “‘전공'/'연구실'/'이메일'/'전화번호'/'강의평'/'강의평가하기' 만 입력해주세요” 라는 응답 반환
    
    2-2. 만약 사용자가 입력한 값이 위의 항목이 아닌 다른 교수명일때
    
     사용자가 응답한 교수명으로 전역변수(교수명)를 재정의 후 해당 교수님의 어떤 정보를 원하는지 질문을 반환
    
    2-3. 만약 사용자가 ‘강의평가하기’를 입력하면 :
    
    2-3-1. 아무것도 입력하지 않으면 “잘못된 강의평입니다.” 응답
    
    2-3-2. 강의평을 입력해주세요 출력 후 감성평가모델을 이용한 함수에 입력받은 강의평을 인수로 입력.
    
      긍정일 때 1, 부정일 때 0을 반환하는 함수에 만들어 반환값이 1이면 좋아요 컬럼에 +1, 0이면 별로에요 컬럼에 +1 (사용자가 ‘강의평’정보를 원할 시 좋아요 x개, 별로에요 y개 입니다. 라는 정보를 반환할 수 있도록) 
    

3. 반복문을 이용하여 응답 이후에도 계속 전역변수에 저장된 교수님의 어떤 정보를 원하는지 질문을 반환
    
    3-1. “없어”를 입력할 시 반복문 종료하여 대화 종료
    

4. Django를 이용하여 제작한 챗봇을 웹에서 구동 

5. 수집한 데이터들을 MySQL DB를 이용해 관리
